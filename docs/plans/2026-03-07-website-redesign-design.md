# Website Redesign - NeutralAI Gateway

## Overview

The website will be a single-page website with a design that evokes trust and security, while still maintaining a modern and clean look. The overall theme will convey a sense of robust protection without overstating compliance maturity.

## Section Elements

The website will consist of the following sections:

1.  **Hero Section**: Reframing the messaging to highlight security from the start.
    *   Headline: "NeutralAI Gateway: Add a Security Control Layer Before Data Reaches External AI"
    *   Brief description of how the Gateway protects sensitive data and supports safer rollout decisions.
    *   Prominent "Request a Demo" call-to-action button.
    *   Visually appealing background image or video conveying security (e.g., a shield protecting data).

2.  **Features Section**: Showcasing key features with a security-focused perspective.
    *   PII Detection Accuracy: Emphasize the precision of PII detection and its role in preventing data leaks.
    *   Integration Simplicity: Highlight how easy it is to integrate the gateway securely into existing systems.
    *   Security Readiness: Describe documented controls, review workflows, and rollout posture without implying certifications not yet held.
    *   Performance & Scalability: Assure potential customers of the robust, secure, and scalable nature of the gateway.

3.  **Benefits Section**: Highlighting benefits with a security angle
    *   Illustrate how NeutralAI Gateway helps businesses to:
        *   Support safer adoption of AI in regulated environments.
        *   Protect sensitive data and uphold customer trust
        *   Improve the accuracy and reliability of LLM applications while maintaining security standards
        *   Minimize the risk of data breaches, security incidents, and regulatory penalties
        *   Save time and resources while maintaining stringent security measures

4.  **Footer**: Standard website footer with accurate trust signals, contact information, social media links, and legal information (privacy policy, terms of service, etc.).
